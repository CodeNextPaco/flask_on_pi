API Documentation

This API contains Code Next coaches' favorite albums. See the useage below.

All Albums

endpoint: /api/v1/albums/all

returns all albums

Search

Search for albums by year, artist and song.

endpoint: /api/v1/albums/search

optional parameters:

  • artist: /api/v1/albums/search?artist=Prince
  • year: /api/v1/albums/search?year=2019
  • song. Will return a whole album if a song matches the string in the query. /api/v1/albums/search?song=Tyrant
